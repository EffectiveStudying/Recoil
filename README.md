# 리코일 스터디

## 계획

리코일의 장단점 확인 후 realworld 프로젝트를 진행하며 코드리뷰

1. 기존 상태관리 라이브러리와 recoil 비교
2. Home page 
3. Sign in, Sign up pages
4. Settings, Profile pages
5. Editor page
6. Article page
